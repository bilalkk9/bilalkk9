<details>
<summary>⚙️ How to enable the snake animation</summary>
<br/>

1. Go to your `bilalkk9/bilalkk9` repository  
2. Click **Actions** → **New Workflow** → **set up a workflow yourself**  
3. Paste the following and save as `snake.yml`:

```yaml
name: Generate Snake
on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: bilalkk9
          outputs: |
            dist/github-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

4. Run the workflow manually once, then it auto-updates every 12 hours!
